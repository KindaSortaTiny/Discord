# Discord Enhancements

This repository contains my personal modifications for Discord.  

## Features

- **IMGView** – Enhances Discord's image viewer.  
- **MemPop** – Makes the Member List pop out instead of always displaying.  
- **Misc** – Various personal fixes and changes:
  - No Nitro Name Effects  
  - Nameplate Visible in Chat  
  - Dark Home Button  
- **SrchPop** – Makes the Search button pop out instead of always displaying.
